# AI Museum Curation System

프로젝트 명
박물관 인공지능 큐레이션 시스템 개발

프로젝트 Abstract
COVID-19 전염병이 확산된 이래, 박물관 접근성이 현저히 떨어지고 있다. 이에 따라 버추얼 뮤지엄과 같은 온라인 기반의 대안이 시도되고 있으나, 큐레이팅 투어를 비롯한 물리적인 장소 안에서 일어나는 문화적 소통과 직접적 체험을 완벽하게 대체할 수 없다는 한계가 존재한다.
본 프로젝트에서는 박물관의 가변적인 환경에서도 방문자들 사이의 거리두기를 감안하고, 더불어 큐레이터의 의도를 반영한 순차적인 작품 감상 경로를 자동적으로 제시하는 ‘인공지능 큐레이션 시스템’을 개발하여 포스트팬데믹 시대에서도 미술관의 사회적 역할 수행 가능성을 제시하고자 한다.

연구목표
게임엔진으로 구현한 박물관 시뮬레이션 환경 내부에서의 AI 큐레이션 알고리즘 개발 및 테스트

연구목표(정량적)
박물관 인공지능 큐레이션 시스템 WebGL 시뮬레이션 1식
KCI 논문 1편
TechArt 논문 1편

AI-CFT 요소
- 인공지능, 게임개발 기술과 예술공학적 관점을 융합하여 박물관이 가지고 있는 미술사적 가치를 보존하는 것을 의의로 둠
- 현실 구조 기반의 시뮬레이션 환경을 게임엔진으로 구현하고, 구성된 환경 내부에서 인공지능 알고리즘을 테스트하여 진행되는 연구 과정을 실시간으로 확인할 수 있게 함
